# Hyderabad AC Services

A modern React and Vite website for Hyderabad AC Services, covering AC service, repairs, installation, maintenance, and exhaust ventilation pages.

## Tech Stack

- React
- Vite
- Framer Motion
- React Icons
- Lucide React
- ESLint

## Available Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
npm run preview
```

## Project Structure

```text
src/
|- assets/
|- components/
|- components/contact/
|- components/blog/
|- data/
|- pages/
`- App.jsx
```

## Notes

The app uses Vite for local development and a lightweight client-side route switch in `App.jsx` for the home, services, blog, and contact pages.
