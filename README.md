# Hyderabad AC Services - Premium React Overhaul

A modern, high-performance redesign of the Hyderabad AC Services platform built using React.js.

Focused on:
- Speed
- Mobile-first UX
- High conversion
- SEO optimization

## Tech Stack

**Core:** React.js (Hooks + Functional Components)  
**Routing:** React Router v6  
**Styling:** Tailwind CSS  
**Animations:** Framer Motion  
**Icons:** Lucide React  
**Forms:** React Hook Form + Zod  
**State:** Zustand

## Key Features

### Bento Service Grid
- Clean responsive layout
- Modern card-based UI
- Smooth hover effects

### Glassmorphic Booking Engine
**Flow:**
1. Select Service
2. Choose Date & Time
3. Confirm via WhatsApp

- Backdrop blur UI
- Multi-step form
- High conversion focused

- Kukatpally

- Secunderabad
- Gachibowli


## Getting Started

### 1. Clone Repo
```bash
git clone https://github.com/your-username/hyd-ac-react.git
```

### 2. Go to Folder
```bash
cd hyd-ac-react
```

### 3. Install Dependencies
```bash
npm install
```

### 4. Run Project
```bash
npm run dev
```
